EDDV - Hannover Airport
----------------------------

How to install the scenery?

Copy or unzip all files and put it in the following folder:
Flightgear -> data -> Scenery -> Objects -> e000n50 -> e009n52

Be sure that you replace the stg-file named "3105690" otherwise it won't work later.
